üõ† Service Methods

The WhiteboardService provides methods to manage elements, control the canvas, and interact with the whiteboard.

üéØ Method üìù Description üõ† Parameters
addElement(element: WhiteboardElement) Adds a new element (e.g., shape, text) to the whiteboard. element: WhiteboardElement
addImage(image: string, x?: number, y?: number) Adds an image to the whiteboard at a specified position. image: string, x?: number, y?: number
removeElements(ids: string[]) Removes elements from the whiteboard by IDs. ids: string[]
updateElement(element: WhiteboardElement) Updates an existing element with new properties. element: WhiteboardElement
updateSelectedElement(partialElement: Partial<WhiteboardElement>) Modifies only specific properties of the currently selected element. partialElement: Partial<WhiteboardElement>
selectElement(element: WhiteboardElement | null) Selects or deselects an element on the whiteboard. element: WhiteboardElement | null
clear() Clears all elements from the whiteboard. none
undo() Reverts the last action. none
redo() Restores the last undone action. none
save(format = FormatType.Base64, name = 'New board') Saves the current whiteboard state in the specified format (e.g., Base64, JSON, SVG). format: FormatType, name: string
setActiveTool(tool: ToolType) Sets the current drawing tool (e.g., pen, eraser, shape). tool: ToolType
setCanvasDimensions(width: number, height: number) Sets the width and height of the whiteboard canvas. width: number, height: number
setCanvasPosition(x: number, y: number) Moves the canvas to a specific position. x: number, y: number
centerCanvas() Centers the whiteboard canvas within the viewport. none
fullScreen() Toggles full-screen mode for the whiteboard. none
toggleGrid() Enables or disables the background grid for alignment. none
