info Multi-Instance Support:

ng-whiteboard supports multiple independent whiteboard instances. When using NgWhiteboardService, you must specify which board you're working with using setActiveBoard(boardId).

The config input accepts a WhiteboardConfig object with the following properties:

Property Type Default Description
drawingEnabled boolean true Enable or disable drawing on the canvas
canvasWidth number 800 Width of the whiteboard canvas in pixels
canvasHeight number 600 Height of the whiteboard canvas in pixels
fullScreen boolean true If true, canvas automatically fits the parent container
center boolean true Center canvas in parent (works with fullScreen: false)
canvasX number 0 X position of canvas (when center is false)
canvasY number 0 Y position of canvas (when center is false)
strokeColor string #333333 Default stroke/border color for drawing
strokeWidth number 2 Default stroke width in pixels
backgroundColor string #F8F9FA Canvas background color
lineJoin LineJoin LineJoin.Round Style of line joins (Round, Bevel, Miter)
lineCap LineCap LineCap.Round Style of line caps (Round, Butt, Square)
fill string transparent Default fill color for shapes
zoom number 1 Zoom level (1 = 100%, 2 = 200%, etc.)
x number 0 Viewport X offset for panning
y number 0 Viewport Y offset for panning
fontFamily string sans-serif Default font family for text elements
fontSize number 24 Default font size in pixels
dasharray string '' SVG dasharray pattern for dashed lines (e.g., '5,5')
dashoffset number 0 SVG dashoffset for dashed line patterns
enableGrid boolean false Show or hide grid pattern on canvas
gridSize number 10 Size of grid cells in pixels
snapToGrid boolean true Snap elements to grid when enabled

signal_cellular_alt Reactive Signals

Use signals(boardId) to get reactive signals for a specific board. These signals update automatically when the data changes.

Signal Description
elements All elements on the whiteboard
selectedElements Currently selected elements
layers All layers
activeLayer The currently active layer
config Current whiteboard configuration
activeTool Currently active drawing tool
canUndo Whether undo is available
canRedo Whether redo is available
selectionBox Current selection box bounds
boundingBox Bounding box of selected elements

label Available Tool Types

Use these ToolType values with selectedTool input or setActiveTool() method:

  • ToolType.Select - Selection tool
  • ToolType.Pen - Freehand drawing
  • ToolType.Eraser - Erase elements
  • ToolType.Hand - Pan the canvas
  • ToolType.Line - Draw straight lines
  • ToolType.Arrow - Draw arrows
  • ToolType.Rectangle - Draw rectangles
  • ToolType.Ellipse - Draw ellipses/circles
  • ToolType.Text - Add text
