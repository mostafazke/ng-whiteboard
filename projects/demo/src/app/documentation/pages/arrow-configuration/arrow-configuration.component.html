link Smart Element Binding

Arrows can automatically bind to other elements on the canvas. When an arrow endpoint is dragged near a connectable element, it snaps to the nearest connection point and creates a persistent binding. When a bound element is moved or resized, all connected arrows update their positions automatically.

Connectable Elements

Element Type Connection Points Details
Rectangle 8 points Top, right, bottom, left (cardinal) + top-left, top-right, bottom-right, bottom-left (corners)
Ellipse 8 points Cardinal directions + 45° positions computed on the ellipse edge
Image 4 points Top, right, bottom, left (from bounding box)
Text 4 points Top, right, bottom, left (from bounding box)
info Non-connectable elements:

Arrows cannot bind to other arrows, lines, or pen strokes. Only shapes with defined boundaries support connection points.

How Binding Works

near_me
Snap Detection

When drawing or dragging an arrow endpoint within 20 pixels of a connectable element, the available connection points are highlighted and the endpoint snaps to the nearest one.

sync_alt
Automatic Updates

Bound arrows automatically recompute their endpoints when the connected element is moved, resized, or rotated. This keeps diagrams consistent without manual adjustment.

link_off
Auto-Detach

When a connected element is deleted, its arrow bindings are automatically detached. The arrow remains on the canvas with its last computed endpoint position.

adjust
Gap Control

Each binding includes an optional gap property (in pixels) that maintains a small distance between the element edge and the arrow endpoint, preventing visual overlap.

Binding Properties

Property Type Description
elementId string ID of the shape the arrow endpoint is bound to
pointId string | null Named connection point (e.g., 'top', 'bottom-left') or null for closest edge
focus number 0–1 position along edge for fine-tuning the connection point
gap number Distance in pixels between the shape edge and arrow endpoint

Visual Feedback

The whiteboard provides real-time visual indicators during arrow drawing and editing:

  • Connection point dots — appear on hoverable shapes when an arrow endpoint is nearby
  • Snap indicator — highlights the connection point that will be used if the user releases
